# wydarzenia_psychiatryczne_on-line (final)

Kalendarz wydarzeń psychiatrycznych w Polsce – online/VOD.

## Dlaczego ten algorytm jest mniej "ślepy"
- Nie wymaga sztywno frazy `psychiatr-` (bo to pomija część neuro/CNS i części szkoleń),
- zamiast tego liczy **score tematyczny** na podstawie list słów-kluczy,
- ma też listę seedów (pewne strony), które zawsze sprawdza.

- Kalendarz: https://julescotard.github.io/wydarzenia_psychiatryczne_on-line/psychiatria.ics
- Lista linków: https://julescotard.github.io/wydarzenia_psychiatryczne_on-line/events.html
